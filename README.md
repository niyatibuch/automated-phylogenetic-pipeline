# Automated Phylogenetic Pipeline

Python-based automated framework for phylogenetic tree construction, visualization, and comparative statistical evaluation.

## Overview

This pipeline integrates multiple sequence alignment, phylogenetic tree inference, visualization, and comparative statistical analysis into a reproducible workflow.

## Features

### Multiple Sequence Alignment
- MAFFT
- MUSCLE
- ClustalW

### Tree Inference
- Maximum Likelihood (IQ-TREE)
- Neighbor Joining (NJ)
- UPGMA
- Maximum Parsimony (MP)

### Comparative Analysis
- Robinson–Foulds distance
- Approximately Unbiased (AU) test
- Kishino–Hasegawa (KH) test
- Shimodaira–Hasegawa (SH) test

## Environment

- Python 3.10+
- Developed and tested using Google Colab
- Google Drive used for persistent storage and resume functionality

## Repository Structure

```text
src/          → pipeline source code
datasets/     → input datasets
results/      → generated outputs
figures/      → manuscript figures
docs/         → manuscript and supplementary files
```

## Reproducibility

All datasets used in the manuscript are provided in the `datasets/` directory.

The repository currently contains:

| Dataset   | Description                       |
| --------- | --------------------------------- |
| Dataset 1 | Animal COI sequences              |
| Dataset 2 | Bacterial 16S rRNA sequences      |
| Dataset 3 | Plant rbcL sequences              |
| Dataset 4 | Ant COI taxa dataset              |
| Dataset 5 | Hymenopteran HSP70 gene sequences |

Representative outputs generated by the pipeline are provided in the `results/` directory, including phylogenetic trees, comparison reports, and summary statistics.

All analyses reported in the manuscript can be reproduced using the source code in `src/` together with the datasets provided in this repository.
